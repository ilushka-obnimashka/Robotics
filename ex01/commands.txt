===1===
Название: less
Описание: пейджер терминала линукс, который показывает содержимое файла по одному экрану за раз. Удобно при работе с большими файламий
Синтаксис: less [опции] путь_к_файлу
Пример: 
$ less -N commands.txt 
Примечание: Утилита загружает не весь большой файл, а обращается к нему страница за страницей, что обеспечивает очень высокую скорость загрузки.

===2===
Название: whoami
Описание: выводит имя текущего пользователя под которым выполняются команды
Синтаксис: whoami
Пример: 
$ ilya-trushkin@eto-computer-ubuntu:~$ whoami
$ ilya-trushkin
Примечание: подобного результата можно добиться при помощи команд : 1) id -un
								    2) echo $USER
===3===
Название: neofetch
Описание: отображение информации о системе, включая имя пользователя, имя хоста, дистрибутив, архитектуру, версию ядра и прочие сведения.
Синтаксис: neofetch [опции]
Пример:
$ neofetch
Примечание: Необходимо установить (sudo apt-get install neofetch)

===4===
Название: mtr 
Описание: инструмент для диагностики сети, который объединяет функциональность программ traceroute и ping. Показывает путь передачи пакетов от локального компьютера до указанного удалённого хоста, а также в реальном времени отображает статистику задержек и потерь пакетов на каждом сетевом узле маршрута. 
Синтаксис: mtr [опции] <хост|IP>
Пример: 
$ mtr google.com
Примечание: Команда выведет таблицу с непрерывным обновлением статистики до прерывания пользователем. Каждая строка которой соответствует одному узлу на маршруте.

===5===
Название: tldr
Описание: Утилита предоставляет упрощённые страницы справки для команд терминала,облегчённая альтернатива классическим man-страницам, созданная сообществом для быстрого и наглядного понимания команд.
Синтаксис: tldr [опции] <команда>
Пример: 
$ tldr less
Примечание: Необходимо установить (sudo snapp install tldr)

===6===
Название: history
Описание: Команда history показывает список ранее введённых команд в текущем терминале и позволяет повторно выполнять их без повторного ввода.
Синтаксис: history [опции]
Пример: 
$ history 10 — покажет последние 10 команд из истории.
Примечание: можно повторно вызвать команду используя !<номер команды>

===7===
Название: !!
Описание: 
Двойной восклицательный знак !! в Bash служит для повторного выполнения предыдущей команды, введённой в терминале.
Синтаксис: !!
Пример: 
$ apt install vim
[error...]
$ sudo !!
$ sudo apt install vim
$ Чтение списков пакетов… Готово

===8===
Название: tmux
Описание: Терминальный мультиплексор, позволяющий создавать, управлять и переключаться между несколькими терминальными сессиями в одном окне.
Синтаксис: tmux [опции]
Пример: tmux new -s mysession

===9===
Название: ^
Описание: Символ каретки ^ в командной строке Bash используется для быстрой замены опечатки в последней выполненной команде.
Пример: 
$ sudo apt intsall vim
$ [error...]
$ ^intsall^install^
$ sudo apt install vim
$ Чтение списков пакетов… Готово

===10===
Название: nl
Описание: нумерует строки
Пример: 
$ cat file.txt | nl 
Примечание: больше всего полезна при использовании ее в качестве аргумента
